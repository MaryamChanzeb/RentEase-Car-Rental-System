# 🚗 RentEase - Car Rental Management System

A complete **Console-Based Car Rental Management System** developed in **C++** using **Object-Oriented Programming (OOP)** concepts.

RentEase allows customers to rent cars, return them, view rental history, and make payments, while administrators can efficiently manage the entire rental system through a password-protected admin portal.

---

# 📌 Project Overview

RentEase is designed to simulate a real-world car rental service. It provides separate functionalities for **Customers** and **Administrators**, ensuring smooth management of cars, rentals, and customer records.

The project demonstrates the practical implementation of:

- Object-Oriented Programming (OOP)
- File Handling
- Data Management
- Menu Driven Interface
- Payment Processing
- Rental History Management

---

# ✨ Features

## 👤 Customer Panel

- 🚘 Rent a Car
- 🔍 Search Car by ID
- 📅 Select Rental Duration
- 💳 Payment Processing
- 🧾 Automatic Rental Receipt Generation
- 📜 View Rental History
- 🔄 Return a Car

---

## 🔐 Admin Panel

Protected by password authentication.

Admin can:

- ➕ Add New Cars
- ❌ Remove Cars
- ✏️ Modify Car Information
- 📋 Display All Cars
- 🔍 Search Cars
- 👥 Manage Customers
- 📜 View Rental History
- ⏰ Check Overdue Rentals
- 📁 Generate Sample Car File

---

# 💳 Payment System

The system includes a simple payment module that:

- Calculates total rental cost
- Accepts customer payment
- Calculates remaining balance
- Displays payment confirmation
- Generates rental receipt

---

# 🧾 Rental Receipt

After every successful rental, the system automatically generates a receipt containing:

- Customer ID
- Car ID
- Rental Duration
- Total Cost
- Rental Date

The receipt is saved as:

```
receipt.txt
```
